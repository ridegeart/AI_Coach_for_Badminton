# AI_Coach_for_Badminton
An implement of AI CUP 2023 [教電腦看羽球 - 台灣第一個結合AI與運動的競賽].

## Badminton Track Detect
- 使用 TrackNet.ipynb
- 輸入影片，輸出每幀球的畫素點位置。

## Court Detect
- 使用court_detect.ipynb
- 輸入圖像，輸出場地的中線與前/後場。

## Pose Detect
- 使用 OpenPose.ipynb
- 輸入影像，輸出每幀裡每個人的關節處位置。(25個編碼)
